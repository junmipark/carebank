# CareBank

## 주제
맞춤형 건강관리 사이트

## 기획 의도 및 목표
- 사용자의 종합 건강 관리를 편리하게 돕는 웹 어플리케이션으로, 다양한 API를 활용하여 건강 관리 서비스를 제공한다. 
- 선호도를 기반으로 맞춤 건강정보를 제공하며, 다이어리 및 건강관리 대시보드로 사용자의 건강 상태를 체크한다.
